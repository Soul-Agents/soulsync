🚀 Getting Started
Clone the repository:

git clone https://github.com/adamzasada/soulsync.git

Navigate to the project directory:

cd soulsync

Install dependencies:

npm install

Start the development server:

npm run dev

Visit http://localhost:3000 to see the magic! ✨
